Raytracer:

This is a project to see how Raytracers can be implemented in Javascript both using HTML Canvas for computation and using pure javascript for computations. So for example a GLSL Raytracer will most likely be implemented in the future using a 3D canvas. However The first component will be a complete raytracer built from the ground up in Javascript using a 2D canvas.
