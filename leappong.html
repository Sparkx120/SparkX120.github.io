LeapPong:

This is a project to try out coding for Leap Motion in Javascript. I figured a game would be best so I considered a few and decided on a pong game. I wrote this over one evening for fun and will submit it to the 3D Jam contest by Leap Motion. Please note this game needs ES6 Arrow functions and Classes to work. Chrome works out of the box and Edge with Experimental Features Javascript turned on work.
